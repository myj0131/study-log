## 1️⃣ 리눅스(Linux)란?
리눅스는 윈도우(Windows)나 맥(macOS) 같은 **운영체제(OS)** 의 한 종류입니다. 하지만 결정적인 차이점이 있습니다.<br>
* **오픈 소스:** 누구나 내부 코드를 볼 수 있고 자유롭게 수정할 수 있습니다.<br>
* **서버의 표준:** 전 세계 대부분의 서버, 클라우드, 그리고 도커(Docker)가 바로 이 리눅스 위에서 돌아가도록 설계되었습니다.<br>
* **다양한 배포판:** 리눅스라는 뿌리는 같지만, 사용 목적에 따라 '우분투(Ubuntu)', 'CentOS', '데비안(Debian)' 등 다양한 버전(배포판)이 존재합니다.
<hr>

## 📂 리눅스 파일 시스템의 특징
리눅스는 윈도우와 파일 관리 방식이 완전히 다릅니다. 이 개념을 알면 나중에 도커 경로 설정할 때 정말 편해집니다.
* **모든 것은 파일이다:** 리눅스에서는 하드웨어 장치조차도 하나의 파일로 취급합니다.<br>
* **최상위 경로는 /(Root):** 윈도우의 C:\ 대신, 리눅스는 모든 것이 / 라는 뿌리에서 시작합니다.<br>
* **홈 디렉토리 ~(Tilde):** 현재 로그인한 사용자만의 개인 공간을 뜻합니다. (예: /home/ubuntu)
