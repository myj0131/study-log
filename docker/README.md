# Docker
이 폴더는 Docker 학습 과정에서 정리한 자료, 예제 코드, 프로젝트를 보관하는 공간입니다.  
기본 개념부터 실습 예제, 프로젝트 구성까지 단계적으로 학습한 내용을 체계적으로 정리합니다.
<hr>

## 1️⃣ Docker란?
도커(Docker)는 쉽게 말해 "내 앱이 잘 돌아가는 실행환경을 통째로 포장해서, 어디서든 똑같이 실행" 가능하게 해주는 도구.

* 예전 방식: "내 PC에서는 되는데 서버에서는 안돼요…"
* 도커 방식: "환경(라이브러리/설정/런타임)을 이미지로 고정 → 어디서 돌려도 동일"

## 2️⃣ 도커가 해결하는 문제들
✅ 환경 통일
* 특정 확장, 특정 설정… 이런 것들이 사람 손으로 맞추면 어긋날 수 있지만
  도커는 이미지(설치된 상태의 스냅샷) 로 고정해서 재현하기 때문에 문제 발생 X
